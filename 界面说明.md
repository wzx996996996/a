# 五子棋游戏界面说明

## 界面布局

程序启动后，你会看到以下界面结构：

```
┌─────────────────────────────────┐
│   当前回合: 黑棋                │  ← 顶部（状态标签）
│   AI状态: 未使用                │
├─────────────────────────────────┤
│                                 │
│         五子棋棋盘              │  ← 中间（棋盘区域）
│      (15x15网格)                │
│                                 │
├─────────────────────────────────┤
│  [重新开始] [🤖 使用AI生成代码] │  ← 底部（按钮区域）
└─────────────────────────────────┘
```

## 按钮说明

1. **重新开始** 按钮
   - 位置：底部左侧
   - 功能：清空棋盘，重新开始游戏

2. **🤖 使用AI生成代码** 按钮
   - 位置：底部右侧
   - 功能：调用DeepSeek API生成Java代码
   - 点击后会触发AI代码生成功能

## 如果看不到按钮

如果看不到"使用AI生成代码"按钮，请：

1. **关闭当前运行的程序**
2. **重新编译并运行**：
   ```bash
   export JAVA_HOME=/Users/wzx/Library/Java/JavaVirtualMachines/openjdk-25/Contents/Home
   export PATH=$JAVA_HOME/bin:$PATH
   export DEEPSEEK_API_KEY="sk-ea182b22c22d403f9d093aaf588c6f98"
   ./run-simple.sh
   ```

3. **检查窗口大小**：确保程序窗口没有被最小化或遮挡

4. **查看底部区域**：按钮位于窗口最底部，包含两个按钮

## 测试AI功能

1. 点击 **"🤖 使用AI生成代码"** 按钮
2. 观察AI状态标签的变化：
   - "AI状态: 正在调用DeepSeek API..."
   - "AI状态: 代码生成成功！查看控制台"
3. 查看终端窗口，查看生成的代码
